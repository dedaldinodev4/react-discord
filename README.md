# Discord Web

## Features
* Sign-In Page
* Sign-Up Page
* Dashboard Page


## Technologies
* Vite
* React
* Typescript
* React-Hook-Form
* Styled-components
* Styled-icons
* React-lottie

## Clone & Open
```
    git clone https://github.com/dedaldinodev4/react-discord.git
    cd react-discord
```
## Install & Run
```
  yarn install
  yarn dev
```
